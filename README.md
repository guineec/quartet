# Quartet - Sinatra + ActiveRecord + MySQL + Rake
A quick start generator for database backed Sinatra apps

## Installation
Install by running
`$ gem install quartet`

## Usage
# Generate app
`$ quartet <app_name>`  

### Serve app
`$ rake s`

### Development console
`$ rake c`
